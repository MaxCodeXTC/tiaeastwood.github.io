About This Website

General Info:

I built the current version on my website using only HTML, CSS & Vanilla JavaScript. My aim was to build a responsive site without relying on any frameworks, in order to improve my basic skills. To begin with, I started with a rough sketch of how I wanted my site to look. I knew that I wanted to have a clean one-page layout, with each section flowing to the next, for a smooth user experience. As a portfolio website, I wanted to make it as easy as possible to view the content.

Responsive Design:

I built with mobile in mind, using flexbox, responsive measurements, and media queries where appropriate, in order for my content to look good on different screen sizes. I tested my site on different devices that I owned, to make sure I was happy with how things looked and worked.

Colour Scheme:

After laying the foundations for the structure and content, I experimented with colour palettes. To keep the design of the site cohesive, I settled on 2 main colours (deep blue and bright pink) with white. The pink is an injection of my personality, but together I feel that these colours compliment each other, while also making a pleasing contrast.

Functionality:

To include some user interactivity, I used a JavaScript library called particles.js to create the floating particles on the front page of my site. When the user hovers their mouse, they can move the particles. The idea behind this was to create a more memorable experience. I used JavaScript on other aspects of my site to improve user experience. For example:
  • This modal pop-up box that you're seeing now. I felt that this was the cleanest way to show more information on the same page.
  • Smooth animated page transitions. When you click the arrows to move to the next section of the site, or if you click the arrow to return to the top of the site, it will transition smoothly instead of abrubtly changing to that section.
